### Measuring Tools for Cooking Classifier

## Description: 
A Flutter app that uses a TensorFlow Lite model to identify/measure cooking items via the device camera and provide measurement-related info (uses TFLite models bundled in assets).

## Features:

- On-device inference: Runs a TFLite model locally (no network).
- Camera input: Real-time camera capture for measurement/recognition.
- Cross-platform: Targets Android, iOS and desktop/web where supported by Flutter plugins included.

## Prerequisites

- Flutter SDK: Install Flutter (stable channel).
- Platform toolchains: Android SDK for Android builds; Xcode for iOS builds (macOS).
- Device/emulator: A physical device or emulator/simulator with camera support.
